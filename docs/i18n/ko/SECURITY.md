# 보안과 개인정보

VibeCast는 로컬 텍스트 흐름에 집중합니다. 음성 인식은 Android 키보드 안에서 이루어지고, VibeCast는 입력창에 쓰인 텍스트만 받습니다.

- 웹 페이지는 표준 `<textarea>`를 사용합니다.
- 마이크 권한을 요청하지 않습니다.
- 오디오를 수신, 전송, 저장하지 않습니다.
- Mac은 사용자 텍스트를 외부 서비스로 보내지 않습니다.
- 진단 로그는 본문, 토큰, 클립보드 내용을 기록하지 않습니다.

페어링 토큰은 Mac이 생성하며 URL에 포함됩니다. 토큰을 다시 생성하면 이전 URL은 무효가 됩니다. 쓰기 전에는 대상, Bundle ID, 프로세스, 세션, revision, 빈도와 크기를 검증합니다.

# 문제 해결

## 페이지가 열리지 않음

Mac과 폰이 서로 접근 가능한 네트워크에 있는지, VibeCast가 실행 중인지, 최신 토큰 주소를 쓰는지 확인하세요.

## 연결되지 않음

URL에 `token=...`이 있는지 확인하고, 토큰을 다시 생성했다면 새 주소로 열어 주세요.

## 손쉬운 사용 미승인

시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용에서 VibeCast를 켭니다. 이미 켜져 있으면 껐다 켜고 VibeCast를 다시 실행하세요.

## 동기화/포커스 실패

폰에서 다시 포커스를 누르고 설정 페이지에서 대상을 다시 테스트하세요. Electron/WebView는 `clipboard_replace`를 시도할 수 있습니다.

## 전송 실패

상태가 동기화됨이 될 때까지 기다리고 `sendShortcut`이 대상 앱의 전송 방식과 맞는지 확인하세요.

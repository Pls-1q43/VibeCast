# الأمان والخصوصية

يركز VibeCast على تدفق النص المحلي. يحدث التعرف على الصوت داخل لوحة مفاتيح Android؛ يستقبل VibeCast النص المكتوب في الصفحة فقط.

- تستخدم الصفحة `<textarea>` قياسيًا.
- لا تطلب إذن الميكروفون.
- لا تستقبل الصوت ولا تنقله ولا تحفظه.
- لا يرسل Mac نص المستخدم إلى خدمات خارجية.
- لا تسجل التشخيصات النص الكامل أو الرموز أو محتوى الحافظة.

ينشئ Mac رمز الاقتران ويضعه في URL. عند إعادة توليد الرمز تصبح العناوين القديمة غير صالحة. قبل الكتابة يتحقق VibeCast من الهدف وBundle ID والعملية والجلسة وrevision والمعدل والحجم.

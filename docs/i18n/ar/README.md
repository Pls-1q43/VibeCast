# VibeCast

حوّل هاتفك إلى لوحة إدخال صوتي نصي عن بُعد لنظام macOS، مصممة خصيصًا لـ Vibe Coding.

[中文](../../../README.md) · [English](../en/README.md) · [日本語](../ja/README.md) · [한국어](../ko/README.md) · [Español](../es/README.md) · [हिन्दी](../hi/README.md) · [العربية](README.md)

يعكس VibeCast النص الذي تنتجه لوحة مفاتيح Android إلى Codex وWorkBuddy وNotion وCodeBuddy أو أي تطبيق Mac تضبطه. يبقى التعرف على الصوت داخل لوحة المفاتيح، بينما ينقل VibeCast النص محليًا وبسرعة وبشكل مقصود.

## الميزات

- لوحة إدخال من الهاتف عبر Android Chrome.
- عكس النص لحظيًا إلى تطبيقات macOS.
- مسودات مستقلة لكل هدف.
- إرسال على مرحلتين بعد تأكيد آخر مزامنة.
- ضبط Bundle ID والتركيز والكتابة والإرسال.
- بلا إذن ميكروفون، بلا معالجة صوت، وسجلات تشخيص منقحة.
- Product, design, code, marketing: All by Codex. Thanks to OpenAI.

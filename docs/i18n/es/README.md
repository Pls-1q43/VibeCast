# VibeCast

Convierte tu teléfono en un panel remoto de voz a texto para macOS, diseñado para Vibe Coding.

[中文](../../../README.md) · [English](../en/README.md) · [日本語](../ja/README.md) · [한국어](../ko/README.md) · [Español](README.md) · [हिन्दी](../hi/README.md) · [العربية](../ar/README.md)

VibeCast refleja en tiempo real el texto que produce tu teclado Android hacia Codex, WorkBuddy, Notion, CodeBuddy o cualquier app de Mac configurada. El reconocimiento de voz permanece en el teclado; VibeCast mueve texto de forma local, rápida y controlada.

## Lo esencial

- Panel de entrada desde Android Chrome.
- Texto en tiempo real hacia apps de macOS.
- Borradores independientes por destino.
- Envío en dos fases tras confirmar la sincronización final.
- Configuración de Bundle ID, foco, escritura y envío.
- Sin permiso de micrófono, sin audio, diagnósticos redactados.
- Producto, diseño, código y marketing: All by Codex. Gracias a OpenAI.

# トラブルシューティング

## ページが開けない

Mac とスマホが同じ到達可能なネットワークにあるか、VibeCast が動作中か、最新のトークン付き URL を使っているか確認します。

## 接続されない

URL に `token=...` があるか確認します。トークンを再生成した場合は新しい URL で開き直します。

## アクセシビリティ未承認

システム設定 → プライバシーとセキュリティ → アクセシビリティで VibeCast を有効化します。既に有効なら一度切り替えて再起動します。

## 同期やフォーカスが失敗する

スマホで再フォーカスし、設定ページで対象を再テストします。Electron/WebView では `clipboard_replace` を試し、範囲確認までは `allowSelectAllReplace=false` にします。

## 送信失敗

状態が同期済みになるまで待ち、送信ショートカットが対象 App と合っているか確認します。

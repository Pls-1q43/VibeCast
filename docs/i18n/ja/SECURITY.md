# セキュリティとプライバシー

VibeCast はローカルなテキストフローに集中します。音声認識は Android キーボード内で行われ、VibeCast が受け取るのは入力欄に書かれたテキストだけです。

- Web ページは標準 `<textarea>` を使います。
- マイク権限を要求しません。
- 音声を受信、送信、保存しません。
- Mac はユーザーテキストを外部サービスへ送りません。
- 診断ログは本文、トークン、クリップボード内容を記録しません。

ペアリングトークンは Mac が生成し、コピーされる URL に含まれます。トークンを再生成すると古い URL は失効します。信頼できるネットワークと自分のデバイスで管理してください。

書き込み前には対象、Bundle ID、プロセス、セッション、revision、頻度、サイズを検証します。
